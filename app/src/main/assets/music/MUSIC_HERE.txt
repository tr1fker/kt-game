Фоновая музыка (assets/music/)
=============================

КУДА КЛАСТЬ ФАЙЛЫ
-----------------
В проекте Android Studio:

  app/src/main/assets/music/

Имена — латиница, цифры, подчёркивания, БЕЗ пробелов и БЕЗ запятых в имени
(запятая используется в сохранении списка).

Поддерживаемые расширения: .ogg  .mp3  .wav

После добавления или переименования файлов нужна пересборка приложения
(файлы из assets попадают в APK).

СВОЙ ФАЙЛ С ТЕЛЕФОНА (БЕЗ ПЕРЕСБОРКИ)
-------------------------------------
В игре: главное меню → «Музыка» → «Добавить трек с устройства».
Выбираешь аудиофайл в системном окне — копия сохраняется только
во внутренней памяти приложения (не в общей «Музыке» телефона).
Пересборка APK не нужна. Удалить такой трек можно кнопкой «✕» в строке.

КАК ВЫБРАТЬ В ИГРЕ
-----------------
Главное меню → кнопка «Музыка».

Отдельно отмечаешь треки для колонки «Меню» (все экраны кроме игры)
и для колонки «Уровень» (только экран прохождения уровня).

• Один отмеченный трек — играет по кругу (зациклен).
• Несколько — играют подряд; после полного прохода списка порядок
  снова случайно перемешивается.

По умолчанию (если настройки ещё не сохраняли):
  меню  → music/bgm_menu.ogg (если файл есть)
  уровень → music/bgm_level.ogg (если файл есть)

РЕКОМЕНДАЦИИ
------------
• Формат .ogg обычно удобнее по размеру APK.
• Держи громкость мастеринга сопоставимой с существующими bgm_*.ogg.
